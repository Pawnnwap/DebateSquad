# 第一部分：逻辑基础（McInerny）

> **核心主张**："逻辑不是一门关于如何赢得争论的学问，而是一门关于如何正确思考的学问。赢得争论的最佳方式，就是让自己的思考比对手更正确。"
> —— D.Q. McInerny

---

## 一、逻辑的前提：清晰的思维

### 1.1 接受现实（Be Attentive to the Facts）

逻辑思维的第一步是**如实接受现实**，而非将自己的想象投射到现实上。

| 常见障碍 | 说明 |
|---------|------|
| **选择性注意** | 只看符合自己立场的信息，忽略矛盾证据 |
| **情绪化感知** | 让情绪扭曲对事实的判断 |
| **先入为主** | 在了解事实之前就已形成结论 |
| **概括过度** | 从个别案例推及全体（"一朝被蛇咬..."） |

**辩论应用**：立论前必须全面研判证据，而非仅搜集支持己方的材料。承认对方论点中的合理成分，才能有效驳斥其不合理的部分。

### 1.2 确认性偏误（Confirmation Bias）

人天然倾向于寻找确认既有信念的信息，排斥挑战既有信念的信息。这是最危险的认知陷阱之一。

**破解方法**：
- 主动寻找反驳己方论点的最强证据（"钢铁人"对方论点）
- 将备赛阶段分为"己方最强论证"和"对方最强论证"两个平行任务

## 二、逻辑四大基本原则

这四条原则是所有逻辑推理的基石。违背任何一条，论证必然崩溃。

### 2.1 同一律（Principle of Identity）

> **一个事物就是它本身。A 即 A。**

**含义**：
- 一个词在同一论证中必须始终保持同一含义
- 改变定义就等于改变了讨论对象

**辩论应用**：
- 质询中最常见的策略——揭示对方在不同语境下偷换了关键词的含义
- 己方立论须在全场保持概念一致性（见[辩论方法论.md §1.3 定义](辩论方法论.md)）

**违背示例**：
> 正方："自由是人类最高价值。"
> 质询："那吸毒者追求的也是自由，支持吗？"
> 正方："那种自由不算真正的自由。"

→ 正方在被逼时偷换了"自由"的定义，违反同一律。

### 2.2 矛盾律（Principle of Contradiction）

> **不可能 A 与非 A 同时为真。一个命题与其否命题不能同时成立。**

**含义**：两个相互矛盾的命题，不可能都是真的。

**辩论应用**：
- **归谬法（Reductio ad Absurdum）**：找出对方论证内部或与其他已有立场之间的矛盾，迫使其修正或放弃论点
- 质询的核心目的之一：找到对方内部矛盾（见[辩论方法论.md §8.2 摸打结](辩论方法论.md)）

**发现矛盾的方法**：
1. 追问对方在极端情境下的立场是否一致
2. 比对对方在不同发言中的表述
3. 援引对方在其他公开场合的矛盾表态

### 2.3 排中律（Principle of Excluded Middle）

> **A 与非 A 之间没有中间状态。对于任意命题，它要么为真，要么为假，没有第三种可能。**

**重要限制**：
- 排中律只适用于**非此即彼的命题**
- 很多辩题的关键词存在程度差异，不适合粗暴二分
- 误用排中律会造成"非黑即白"谬误（False Dilemma）

**辩论应用**：
- 当对方试图用"非此即彼"框定己方时，需识别并指出中间地带的存在
- 但在质询时，可将己方有利的命题引向排中律适用的形态，逼对方表态

### 2.4 充足理由律（Principle of Sufficient Reason）

> **任何事物的存在，任何命题的为真，都必须有充足的理由。没有理由，不能接受任何命题。**

**含义**：
- 主张需要证据支撑，断言不等于论证
- 论证的强度取决于支撑理由的质量和数量

**辩论应用**：
- 驳论最基础的技术：指出对方"主张无据"（Bare Assertion）
- 自身立论务必遵守：每个论点都要有对应的论据层级（见[辩论方法论.md §3.2 多层次论证](辩论方法论.md)）

## 三、命题：逻辑的基本单位

### 3.1 命题的定义

**命题（Proposition）**：一个可以判断真假的陈述。
- "今天下雨。" → 命题（可判断真假）
- "请关窗。" → 非命题（祈使句）
- "天哪！" → 非命题（感叹句）

**辩论的核心**：辩题本身是一个命题；正方主张其为真，反方主张其为假或存疑。

### 3.2 命题的四种基本类型（亚里士多德分类）

| 类型 | 符号 | 形式 | 示例 |
|------|------|------|------|
| 全称肯定 | A | 所有 S 是 P | "所有 AI 都会犯错" |
| 全称否定 | E | 没有 S 是 P | "没有 AI 是有意识的" |
| 特称肯定 | I | 有些 S 是 P | "有些 AI 比人类更准确" |
| 特称否定 | O | 有些 S 不是 P | "有些 AI 不能理解语境" |

**辩论应用**：
- 识别对方命题的量词（"所有""有些""大多数"），精准反驳
- 全称命题（A/E）只需一个反例即可推翻
- 特称命题（I/O）不能用单一反例推翻

### 3.3 模态命题

| 模态 | 含义 | 辩论中的意义 |
|------|------|------------|
| **必然性**（必须如此） | 不可能为假 | 最强的主张，最难证明 |
| **可能性**（可能如此） | 不必然为假 | 较弱的主张，更易防守 |
| **偶然性**（恰好如此） | 既非必然也非不可能 | 纯经验事实，需证据支撑 |

**辩论应用**：警惕对方将"偶然性"命题伪装成"必然性"命题。例如："AI 伴侣会给人带来伤害"（偶然）≠ "AI 伴侣必然给人带来伤害"（必然）。

## 四、论证的结构

### 4.1 论证的三要素

```
前提（Premises）→ 推理过程（Inference）→ 结论（Conclusion）
```

| 要素 | 定义 | 辩论对应 |
|------|------|---------|
| **前提** | 用于支撑结论的已知命题 | 论据（数据、案例、定义） |
| **推理** | 从前提到结论的逻辑步骤 | 论证方式（演绎/归纳/类比） |
| **结论** | 通过推理得出的新命题 | 论点（主要论点或子论点） |

### 4.2 有效性 vs 真实性

这是逻辑学中最重要的区分之一：

| 概念 | 含义 |
|------|------|
| **有效（Valid）** | 推理形式正确：如果前提真，则结论必然真 |
| **可靠（Sound）** | 有效 + 所有前提实际上都是真的 |
| **真实（True）** | 对应现实 |

**关键规则**：
- 一个论证可以是**有效但不可靠**（前提假，但推理形式正确）
- 一个论证可以有**真实的前提和结论**，但推理**无效**
- 只有**可靠的论证**才是完整的好论证

**辩论应用**：
- 驳论策略 1：**攻击前提**（前提错误 → 论证不可靠）
- 驳论策略 2：**攻击推理**（推理无效 → 结论不能从前提中导出）
- 攻击推理比攻击前提更难，也更致命

## 五、演绎推理

### 5.1 三段论（Syllogism）

最经典的演绎推理形式：

```
大前提（Major Premise）：所有 M 是 P
小前提（Minor Premise）：S 是 M
结论（Conclusion）：   因此 S 是 P
```

**经典例子**：
```
大前提：所有人都会死
小前提：苏格拉底是人
结论：  苏格拉底会死
```

### 5.2 三段论的有效规则（McInerny 总结）

| 规则 | 说明 |
|------|------|
| 三段论只能有三个词项 | 中项必须是同一概念 |
| 中项必须在前提中至少周延一次 | 中项不周延 → 无效 |
| 在结论中周延的词项必须在前提中也周延 | 否则"非法扩大" |
| 两个否定前提不能得出结论 | 否则无效 |
| 一个否定前提只能得出否定结论 | 否则无效 |

### 5.3 假言推理（Hypothetical Syllogism）

辩论中最常用的演绎形式：

**肯定前件（Modus Ponens）**——有效：
```
如果 P，则 Q
P 成立
因此 Q 成立
```

**否定后件（Modus Tollens）**——有效：
```
如果 P，则 Q
Q 不成立
因此 P 不成立
```

**否定前件**——无效（常见谬误）：
```
如果 P，则 Q
P 不成立
因此 Q 不成立    ← 错误！Q 可能由其他原因成立
```

**肯定后件**——无效（常见谬误）：
```
如果 P，则 Q
Q 成立
因此 P 成立    ← 错误！Q 可能由其他原因成立
```

**辩论应用**：质询时将对方论点拆解为假言推理形式，再检验其是否犯了"否定前件"或"肯定后件"谬误。

## 六、归纳推理

### 6.1 归纳推理的性质

归纳推理从**特殊到一般**，结论的确定性低于演绎推理，只能得到**或然性**结论。

```
样本观察 → 归纳概括 → 普遍结论（概率性）
```

### 6.2 强归纳 vs 弱归纳

| 类型 | 条件 |
|------|------|
| **强归纳** | 样本大、随机、多样；前提真 → 结论极可能真 |
| **弱归纳** | 样本小、有偏差；前提真 → 结论仅略可能真 |

### 6.3 类比推理（Analogical Reasoning）

类比是归纳推理的特殊形式：通过两个事物的相似之处，推断其在另一方面也相似。

**类比强度取决于**：
1. 相似点的数量
2. 相似点的相关性（与待推导的性质越相关越好）
3. 两者差异的大小
4. 是否有负面类比案例

**辩论应用**：
- 类比是华语辩论最常用的说服工具（见[辩论方法论.md §8.5 攻击不当类比](辩论方法论.md)）
- 攻击对方类比：找到两者最关键的**不相似点**，证明类比失效
- 建立己方类比：选择与己方论点最相关维度高度吻合的类比

## 七、谬误分类

谬误（Fallacy）：外表像论证，但实质上论证无效或不诚实的推理。

### 7.1 形式谬误（Formal Fallacies）

**推理形式本身错误**，无论前提真假，结论都不能从前提中合法推出。

| 谬误 | 形式 | 示例 |
|------|------|------|
| **否定前件** | 若 P 则 Q；非 P；故非 Q | "不经历苦难就不会成长；他没有苦难经历；故他没有成长" |
| **肯定后件** | 若 P 则 Q；Q；故 P | "好人才善良；他很善良；故他是好人" |
| **中项不周延** | 所有 A 是 B；所有 C 是 B；故所有 A 是 C | "所有鱼会游泳；所有鸭子会游泳；故所有鱼是鸭子" |

### 7.2 非形式谬误（Informal Fallacies）

推理形式可能正确，但内容存在误导。分三大类：

#### 7.2.1 相关性谬误（Fallacies of Relevance）

前提与结论实际上**无关**，但表面上看起来相关。

| 谬误名称 | 定义 | 辩论示例 |
|---------|------|---------|
| **人身攻击（Ad Hominem）** | 攻击人而非论点 | "他说 AI 无法有情感，但他本人也像机器人一样冷漠" |
| **诉诸权威（Appeal to Authority）** | 引用权威作为论据，但权威与论题无关 | "著名物理学家说 AI 伴侣不道德，所以它就是不道德的" |
| **诉诸大众（Ad Populum）** | 以多数人的观点作为论据 | "大多数人都觉得 AI 伴侣很可怕，所以它是可怕的" |
| **诉诸怜悯（Appeal to Pity）** | 用激发同情取代论证 | "我们应该支持 AI 伴侣，因为孤独老人太可怜了" |
| **诉诸恐惧（Appeal to Fear）** | 用恐吓替代论证 | "如果允许 AI 伴侣，人类文明就要终结了" |
| **稻草人（Straw Man）** | 曲解对方立场，然后攻击被曲解的立场 | 对方：AI 伴侣应有界限；攻击：他们认为人类不需要朋友 |
| **红鲱鱼（Red Herring）** | 引入无关话题转移注意力 | 讨论 AI 伴侣葬礼时转向 AI 版权问题 |

#### 7.2.2 预设谬误（Fallacies of Presumption）

论证中**预设了未经证明的前提**。

| 谬误名称 | 定义 | 辩论示例 |
|---------|------|---------|
| **复杂问题（Complex Question）** | 在问题中预设未被确认的前提 | "你什么时候停止逃避这个问题？"（预设对方一直在逃避） |
| **假二分法（False Dilemma）** | 虚构只有两个选项，但实际上还有其他选项 | "要么支持 AI 伴侣，要么你反对科技进步" |
| **循环论证（Begging the Question）** | 用结论作为支撑前提的理由 | "AI 伴侣没有意义，因为它只是程序，程序没有真正意义" |
| **滑坡（Slippery Slope）** | 断言 A 必然导致 Z，但中间的每一步都未被证明 | "给 AI 伴侣举行葬礼，下一步就要给烤面包机立碑" |
| **以偏概全（Hasty Generalization）** | 从不充分的样本得出普遍结论 | "我认识的三个用 AI 伴侣的人都很孤僻，所以 AI 伴侣会让人孤立" |
| **轻率类比（False Analogy）** | 两个事物在关键维度上不相似 | "AI 伴侣就像假花，只是装饰品" |

#### 7.2.3 语言谬误（Fallacies of Ambiguity）

利用语言的**模糊性**进行误导。

| 谬误名称 | 定义 | 辩论示例 |
|---------|------|---------|
| **歧义（Equivocation）** | 同一词在论证中使用不同含义 | "人类有权利（right）做任何自然的事；AI 是人类创造的，是自然的（natural）；故人类有权利使用 AI 伴侣"（偷换 natural 含义） |
| **合成谬误（Composition）** | 将部分的属性归于整体 | "每个辩手都很厉害，所以这支队伍一定很强" |
| **分解谬误（Division）** | 将整体的属性归于部分 | "这支队伍是冠军队，所以每个辩手都是最强的" |
| **重音谬误（Accent）** | 通过强调不同的词改变意思 | "**你**不应该打他"（言下之意别人可以） |

# 第二部分：论辩学理论（Hollihan & Baaske）

> **核心主张**："论辩不只是争吵或辩论，它是人类在面对不确定性时共同寻找最佳答案的理性方法。"
> —— Hollihan & Baaske

---

## 八、论辩的本质与功能

### 8.1 什么是论辩（Argument）

Hollihan & Baaske 将论辩定义为：

> **论辩是一种通过提出理由和证据来影响他人信念或行动的行为。它发生在存在不确定性（uncertainty）的领域。**

关键特征：
- **争议性（Controversiality）**：只有在存在合理分歧时，才有论辩的意义
- **目的性（Purposiveness）**：论辩旨在说服，而非表达情绪
- **理性（Rationality）**：依赖推理和证据，而非强制或欺骗

### 8.2 论辩的四大功能

| 功能 | 说明 |
|------|------|
| **决策功能** | 帮助个人和团体在不确定性中做出更好的选择 |
| **检验功能** | 通过对抗性检验，找到立场中的漏洞和缺陷 |
| **社会功能** | 通过理性说服替代权力强制，维护民主决策 |
| **认知功能** | 促使论辩双方深化对问题的理解 |

### 8.3 辩论语境中的论辩特征

学术辩论（Academic Debate）是论辩的特殊形式，有以下特征：

| 特征 | 说明 |
|------|------|
| **人为对立** | 辩手可能并不真正持有己方立场，但须严格维护它 |
| **明确规则** | 时间、顺序、角色均有明确规定 |
| **第三方评判** | 说服对象是评委，而非对方辩手 |
| **竞技性** | 有输赢，影响参与者的论辩策略 |

## 九、图尔明论证模型

英国哲学家 Stephen Toulmin 提出的论证分析模型，是现代论辩学的核心工具。

### 9.1 基本六要素

```
数据（Data/Grounds）
    ↓ 经由
保证（Warrant）
    ↓ 以
支撑（Backing）为据
    ↓ 得出（在限定词下）
主张（Claim）
    ↑ 除非有
反驳（Rebuttal）
```

| 要素 | 定义 | 示例（AI伴侣辩题）|
|------|------|----------------|
| **主张（Claim）** | 论辩者试图建立的核心命题 | "应该邀请人类出席 AI 伴侣的葬礼" |
| **数据（Data）** | 支撑主张的事实、证据、案例 | "超过 60% 的 AI 伴侣使用者表示从中获得了真实的情感慰藉" |
| **保证（Warrant）** | 连接数据和主张的推理规则 | "提供了真实情感价值的关系值得被认可和纪念" |
| **支撑（Backing）** | 支持保证本身的理由 | "社会学研究表明，悼念仪式是处理情感关系失去的有效机制" |
| **限定词（Qualifier）** | 说明主张成立的条件和程度 | "在使用者有真实情感依附的情况下" |
| **反驳（Rebuttal）** | 主张不成立的例外情况 | "除非该 AI 伴侣从未建立实质性关系" |

### 9.2 图尔明模型的辩论应用

**攻击策略**：对应攻击图尔明六要素中的任何一个。

| 攻击目标 | 攻击方式 | 示例 |
|---------|---------|------|
| 攻击**数据** | 质疑证据的可靠性、来源、样本量 | "60% 这个数字来自哪里？样本是多少？" |
| 攻击**保证** | 质疑推理规则是否普遍成立 | "提供情感价值的关系就值得葬礼仪式？那心理咨询师怎么算？" |
| 攻击**支撑** | 质疑保证的背景依据 | "那项社会学研究是否适用于人机关系？" |
| 攻击**限定词** | 证明实际情况不满足主张成立的条件 | "大多数 AI 伴侣使用者并未建立真实的情感依附" |
| 承认**反驳** | 援引例外情况扩大其适用范围 | "对方承认的例外情况，在现实中是常态" |

## 十、主张的三种类型

Hollihan & Baaske 将辩论主张（Claim）分为三类，每类需要不同的论证策略。

### 10.1 事实性主张（Claims of Fact）

> 主张某件事是否存在、是否发生、或是否属实。

**辩证关键**：提供可核实的证据，建立合理的推断链。

| 要求 | 说明 |
|------|------|
| **可核实性** | 主张必须原则上可被事实所检验 |
| **证据层级** | 直接证据 > 间接证据 > 合理推断 |
| **因果关联** | 区分"相关"与"因果"，避免以相关冒充因果 |

**辩论示例**：
- "AI 伴侣能够感知用户情绪。"（存在性主张）
- "使用 AI 伴侣的老人社交孤立程度更低。"（关联性主张）

### 10.2 价值性主张（Claims of Value）

> 主张某事物的好坏、优劣、对错、应否等价值判断。

**辩证关键**：建立评价标准，并证明己方立场在该标准下更优。

| 步骤 | 操作 |
|------|------|
| **建立评判标准** | 明确用什么维度评价（如：人类尊严/社会福祉/个人自主） |
| **确立标准优先级** | 当标准冲突时，哪个应该优先（需额外论证） |
| **应用标准** | 证明在该标准下，己方立场确实更优 |

**辩论示例**：
- "邀请人类出席 AI 伴侣葬礼是有意义的。"
- "人机关系的情感价值不低于人际关系。"

**注意**：价值性主张常依赖价值层级的共识。若双方持有不同价值体系，需先争夺"用哪个标准评判"。

### 10.3 政策性主张（Claims of Policy）

> 主张某种行动或政策应当被采取。

**辩证关键**：同时需要事实论证（现状有问题/解决方案有效）和价值论证（解决方案符合价值导向）。

| 论证维度 | 问题 |
|---------|------|
| **现状（Stock Issues）** | 现在是否存在需要解决的问题？ |
| **害处（Harm）** | 该问题造成的危害有多大、多广、多严重？ |
| **根因（Inherency）** | 问题是否由现行体制/规则造成？（对应[辩论方法论.md §1.4 根属性](辩论方法论.md)） |
| **解决（Solvency）** | 提案是否能有效解决问题？ |
| **益损比（Advantage/Disadvantage）** | 提案的好处是否大于带来的新问题？ |

**辩论示例**：
- "社会应为 AI 伴侣的葬礼赋予法律认可。"
- "AI 公司应提供情感依附转移支持服务。"

## 十一、证据与支撑

### 11.1 证据的类型

| 类型 | 定义 | 优点 | 缺点 |
|------|------|------|------|
| **实例（Example）** | 具体事件或案例 | 直观、有情感共鸣 | 可能非典型 |
| **统计（Statistics）** | 量化数据 | 客观、可量化 | 可被操纵、需要解读 |
| **证词（Testimony）** | 权威人士或亲历者陈述 | 权威性强 | 取决于来源可信度 |
| **类比（Analogy）** | 与已知事物的比较 | 易理解 | 可能存在关键差异 |

### 11.2 证据评估五维度（Hollihan & Baaske）

| 维度 | 问题 |
|------|------|
| **近时性（Recency）** | 这个数据/案例有多新？ |
| **相关性（Relevance）** | 证据是否真的支持主张？ |
| **充分性（Sufficiency）** | 证据的数量和质量是否足够？ |
| **代表性（Representativeness）** | 案例是否典型，还是极端个例？ |
| **来源可信度（Source Credibility）** | 证据来源是否权威、无偏见？ |

### 11.3 因果推理的常见陷阱

| 陷阱 | 说明 |
|------|------|
| **后此谬误（Post Hoc）** | A 在 B 之前发生 → A 是 B 的原因（错误）|
| **多因混淆** | 将单一因素当作结果的唯一原因 |
| **以相关为因果** | 两者同时变化 ≠ 存在因果关系 |
| **忽略中介变量** | A 和 B 都由 C 引起，但误以为 A 引起 B |

## 十二、驳论与反驳

### 12.1 驳论的四种基本路径

Hollihan & Baaske 将驳论方法归纳为四类：

| 路径 | 操作 | 使用时机 |
|------|------|---------|
| **直接否定（Direct Refutation）** | 提供相反证据或论证，直接反驳对方主张 | 有充分反证时 |
| **减弱（Diminishment）** | 承认对方论点部分成立，但论证其重要性/规模不足以支撑结论 | 无法完全否定时 |
| **转化（Turning）** | 证明对方的证据或论点实际上支持己方结论 | 对方证据对己方有利时 |
| **包容（Absorption）** | 将对方论点纳入己方论证框架，转化为己方优势 | 需要展示大局观时 |

### 12.2 有效驳论的五步框架

```
1. 复述（Restate）   → 准确陈述对方论点（避免稻草人）
2. 回应（Respond）   → 明确表态：承认/否认/质疑
3. 推理（Reason）    → 解释为何对方论点不成立或不足够
4. 证据（Resource）  → 提供支撑己方驳论的证据
5. 结论（Result）    → 说明驳倒此论点后的论证影响
```

### 12.3 战略性反驳决策

并非所有对方论点都值得反驳。驳论的资源（时间、字数）有限，需要战略分配：

| 决策维度 | 问题 |
|---------|------|
| **重要性** | 这个论点对对方立论的成立有多关键？ |
| **可驳性** | 我方有足够的证据和逻辑来有效驳斥吗？ |
| **机会成本** | 花时间驳斥此点是否值得，还是应该推进己方论证？ |
| **受众影响** | 评委对此论点的认同程度如何？ |

**原则**：优先驳斥对方**最核心**且**我方最有把握**驳倒的论点。（对应[辩论方法论.md §10.3 核心错误之睚眦必报](辩论方法论.md)）

## 十三、辩题分析与论证负担

### 13.1 辩题分析（Case Analysis）

Hollihan & Baaske 提供的系统性辩题分析框架：

**Step 1：识别争议（Identifying the Issue）**

> 辩题的争议焦点是什么？双方的核心冲突在哪里？

- 将辩题分解为若干**争点（Issues）**
- 每个争点是双方必须在此形成明确对立观点的小命题
- 争点 ≠ 论点；争点是双方共同承认需要解决的问题

**Step 2：确定论证负担（Burden of Proof）**

| 负担 | 说明 |
|------|------|
| **举证责任（Burden of Proof）** | 谁主张谁举证；正方通常负主要举证责任 |
| **论证负担（Burden of Clash）** | 双方都必须直接回应对方的核心论点 |
| **满足负担（Burden of Rejoinder）** | 一方论证后，另一方有义务回应，否则视为默认 |

**Step 3：定义关键词（Defining Terms）**

- 定义框定讨论范围
- 定义争夺本身是辩论的核心战场之一
- 好的定义应当：合理、明确、不预设结论

### 13.2 辩题五要素分析（Hollihan & Baaske 原版）

| 要素（英文） | 说明 | 辩论对应 |
|------------|------|---------|
| **Significance（重要性）** | 辩题所涉及的问题有多重要？ | 论证问题的规模和严重性 |
| **Inherency（根属性）** | 问题是当前体制固有的？ | 证明需要改变现状 |
| **Solvency（解决力）** | 己方方案能否有效解决？ | 政策型辩题的核心 |
| **Advantages（益处）** | 己方方案带来什么好处？ | 正方核心论证 |
| **Disadvantages（害处）** | 己方方案带来什么坏处？ | 反方核心论证 |

# 第三部分：理论在辩论实战中的映射

## 综合应用指南

将两书理论与华语辩论实战联系起来：

### 备赛阶段

| 任务 | McInerny 工具 | Hollihan & Baaske 工具 |
|------|--------------|----------------------|
| 分析辩题 | 识别辩题的命题类型（事实/价值/规范） | 辩题五要素分析；识别争点 |
| 建立立论 | 演绎/归纳框架；检查三段论有效性 | 图尔明模型；主张类型选择 |
| 预判攻防 | 谬误分类表（识别对方可能用的谬误） | 驳论四路径；战略反驳决策 |
| 搜集证据 | 充足理由律 | 证据五维度评估 |

### 台上阶段

| 场景 | McInerny 工具 | Hollihan & Baaske 工具 |
|------|--------------|----------------------|
| 质询 | 找形式谬误；同一律检验（偷换概念） | 图尔明模型攻击（攻击数据/保证） |
| 驳论 | 有效性 vs 真实性区分 | 驳论五步框架 |
| 自由辩论 | 矛盾律应用（找对方内部矛盾） | 减弱/转化路径 |
| 总结陈词 | 充足理由律（总结证据链） | 战略优先选择最关键争点 |

### 常见逻辑弱点速查（辩论版）

| 弱点名称 | 快速识别 | 快速反驳 |
|---------|---------|---------|
| 人身攻击 | "他这个人..." | "人品与论点无关，请回应论点本身" |
| 假二分法 | "要么...要么..." | "还有第三种可能：..." |
| 稻草人 | 对方夸大或曲解己方 | "我方从未说过...，我方的实际立场是..." |
| 滑坡谬误 | "A 会导致 B 会导致 C..." | "每一步推论需要独立证据，不能假设必然发生" |
| 后此谬误 | "A 之后 B 发生，所以 A 导致 B" | "时间先后不等于因果，请提供机制说明" |
| 诉诸权威 | "专家说..." | "请问该专家在此具体问题上的专业资质？" |
| 循环论证 | 用结论证明前提 | "您用结论来证明前提，这是循环推理" |
| 以偏概全 | 用少数案例推全体 | "样本数量？代表性如何？" |
