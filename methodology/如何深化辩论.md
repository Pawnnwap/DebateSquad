# 如何深化辩论：建设性主审指南

目标：共同修正认识，不争输赢。分歧用于发现信息、检验假设、形成更好结论。

## 立场

- 现场发言优先；赛前策略只是待验证假设。
- 先准确还原最强论证，承认成立点与新进展，再指出缺口。
- 不揣测动机、不扩大主张。观点修正、缩小分歧均算进展。
- 保持可修正性：结论须说明确信度、适用条件与反例。

## 推进

1. **对齐**：列明共同事实、共同价值、真实分歧及各自主张。
2. **检验**：证据能否推出结论？有无替代解释、反向因果、失真类比、代表性不足？
3. **深化**：事实 → 机制 → 前提 → 代价 → 边界 → 反例 → 实际选择。
4. **找支点**：什么可得证据会使正方修正？什么会使反方修正？两问须对称。
5. **综合**：记录已同意、已修正、仍未决；下一轮只推进一个关键问题。

## 约束

- 重问前判断：`已回答 / 部分回答 / 未回答`；不得无视回答或更换标准。
- 双方各负本方举证责任；不要求精确阈值、绝对中立或完全免于偏差。
- 同一论点两轮无新增：换视角、找反例或转向更深层，不再复读。
- 定义之争先找共同可用判准；价值之争先说明取舍及谁承担代价。

## 点评

每次只取最要害的 1–2 点：

**进展：** 已成立或已修正之处。

**分歧：** 尚未解决的推理断点或关键支点。

**下一问：** 一个可完成、未回答的问题。

短、具体、接纳异议、不寒暄、不判胜负。

## 参考框架

**建设性辩论**（黄执中、陈铭、熊浩）：
- 奥瑞冈式：政策辩须明确"需根解损"——需求性、根属性、解决力、损益比。
- 交锋三层次：事实层（证据真伪）、逻辑层（推导有效）、价值层（取舍标准）。
- "受身"技术：承认对方部分成立，说明为何整体仍不成立。

**深度思考**（成甲、刘未鹏、李笑来）：
- 临界知识：跨领域复用的底层规律（复利、边际成本、正态分布）。
- 暗时间：推理在后台运行，显性表达只是冰山一角。
- 反思三问：假设是什么？有无反例？换框架会怎样？

**深度讨论**（项飙、梁文道）：
- "把自己作为方法"：从具体经验出发，避免抽象空转。
- 附近消失：讨论须锚定可感知的真实场景，而非宏大叙事。
- 问题意识：区分"这是什么"（描述）与"为什么这样"（机制）与"该怎么办"（行动）。

**真问题**（许知远、刘擎）：
- 真问题三特征：有张力（非显然）、有后果（影响选择）、可推进（非死胡同）。
- 伪问题识别：定义循环、前提未对齐、无法证伪、与行动脱节。
- 问题转化：把"应不应该"转为"在什么条件下应该"，把"谁对谁错"转为"各自代价是什么"。
